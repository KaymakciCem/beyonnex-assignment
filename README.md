# Beyonnex assignment

This project helps users to find the anagrams and it can provide the anagrams for the given stirng


### Tech Stack

---
- Java 17
- Spring Boot
- JUnit 5

## Run & Build

first go to the terminal and open up the project directory. "~/beyonnex-assignment-master/"

### build the project

./gradlew clean build

### Run the tests

./gradlew test

### Run project

./gradlew bootrun
